# Calorie Counter 

A simple web app to track your daily calorie intake.

## Features
- Add food items with their calorie count
- Remove individual food items
- View running total calories
- Reset list for a new day
- Data persists after page refresh using localStorage

## Project Structure
```
calorie-counter/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Getting Started
1. Clone the repository
2. Open `index.html` in your browser

## Tech Stack
- HTML5
- CSS
- JavaScript
- localStorage
